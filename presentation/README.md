# Presentation Export

Open `presentation/index.html` in Chrome (or any modern browser) and use Print → Save as PDF.

Recommended settings (Chrome):

- Destination: `Save as PDF`
- Layout: `Landscape`
- Paper size: `A4` or `Letter` (choose based on your need)
- Margins: `Default` or `None` (if you want edge-to-edge)
- Scale: `100` (adjust if content needs fitting)
- Check `Background graphics` to include colors and styles

Alternative: use `puppeteer` or `wkhtmltopdf` to programmatically export slides.

Replace placeholder text or add screenshots by editing `index.html`.
